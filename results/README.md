# 📊 Results & Performance Analysis

This directory contains the training outputs, evaluation metrics, dataset visualizations, and prediction samples generated during the development of the **Real-Time Helmet Detection System using YOLOv8**.

---

## 🎯 Project Objective

The objective of this project is to automatically detect whether a rider is wearing a helmet or not using a custom-trained YOLOv8 object detection model.

The model was trained on annotated rider images and evaluated using industry-standard object detection metrics.

# 🚀 Model Performance

## Evaluation Metrics

| Metric              | Score |
| ------------------- | ----- |
| AP (With Helmet)    | 0.877 |
| AP (Without Helmet) | 0.654 |
| mAP@0.5             | 0.765 |

These results indicate strong detection performance, particularly for helmet-wearing riders.

# 💡 Key Observations

* Strong performance on clearly visible riders.
* Reliable helmet detection across different viewpoints.
* Good generalization capability on unseen validation images.
* Performance decreases in heavily occluded or low-resolution scenarios.
* The model successfully learns discriminative visual features associated with helmet usage.

---

# 🏁 Conclusion

The experimental results demonstrate that YOLOv8 is an effective framework for real-time helmet detection tasks. The trained model achieves reliable detection performance while maintaining fast inference speeds, making it suitable for practical computer vision applications and road safety monitoring research.
